# 📱 Tour Booking App

A simple and beginner-friendly **Tour Booking Mobile Application** built using Flutter.
This app allows users to view tour packages, check details, and book tours easily.

---

## 🚀 Features

* 🏝️ View available tour packages
* 📄 See tour details (location & price)
* 📝 Book a tour by entering user name
* ✅ Booking confirmation popup
* 📱 Clean and simple UI


## 🛠️ Tech Stack

* Flutter (UI Framework)
* Dart (Programming Language)

## ⚙️ Installation & Setup

1. Install Flutter SDK

2. Clone the repository:

   git clone https://github.com/your-username/tour-booking-app.git

3. Navigate to project folder:

   cd tour-booking-app

4. Install dependencies:

   flutter pub get

5. Run the app:

   flutter run


## 📱 App Screens

* Home Screen → List of tours
* Detail Screen → Tour information
* Booking Screen → Enter name & confirm
